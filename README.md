# Team Work Website - Front-end Project

هذا المشروع هو تحويل تصميم **Figma** لموقع "Team Work" إلى موقع ويب متكامل ومتجاوب، وهو موقع متخصص في تقديم الخدمات البرمجية والحلول الرقمية.

## 1. نظرة عامة على المشروع (Overview)
الموقع يهدف إلى عرض خدمات الشركة (مثل تطوير الويب، تصميم الشعارات، كتابة مقالات وأبحاث، تحليل البيانات، برمجة التطبيقات، Flutter) وأعمالها السابقة بأسلوب عصري وجذاب، مع دعم اللغتين العربية والإنجليزية.

## 2. التقنيات المستخدمة (Tech Stack)
* **HTML5:** لبناء الهيكل الأساسي للموقع.
* **CSS3 (Custom Properties & Flexbox):** للتنسيق وضمان مرونة التصميم.
* **Vanilla JavaScript:** لإضافة التفاعل ونظام الترجمة (i18n) بدون مكتبات خارجية.
* **Bootstrap 3:** لضمان استجابة الموقع لكافة الشاشات (Responsive Design).
* **Google Fonts:** لاستخدام الخطوط المناسبة الموضحة في التصميم.

## 3. نظام تعدد اللغات (i18n System)
يحتوي المشروع على نظام ترجمة مخصص يتيح تبديل اللغة بسلاسة:
* **حفظ التفضيلات:** يتم تخزين لغة المستخدم في `localStorage` لضمان بقائها عند تحديث الصفحة.
* **دعم RTL/LTR:** يتغير اتجاه الصفحة تلقائياً عند التبديل بين العربية والإنجليزية.
* **ملفات الترجمة:** يعتمد النظام على ملفات JSON موجودة داخل مجلد `locales/`.

## 4. تنظيم وهيكلة الملفات
تم تنظيم ملفات المشروع بشكل يسهل الوصول إليها وإدارتها، وهي كالتالي:
```text
/team-work-website
│
├── /css             # ملفات التنسيق (مثل style.css)
├── /fonts           # ملفات الخطوط المستخدمة في التصميم
├── /images          # الصور، الأيقونات، وشعار الشركة
├── /js              # ملفات البرمجة التفاعلية ونظام الترجمة (i18n.js)
├── /locales         # ملفات الترجمة (ar.json, en.json)
├── index.html       # الصفحة الرئيسية للموقع
└── README.md        # ملف التوثيق الحالي
```

## 5. أقسام الموقع الأساسية (Key Sections)
* **Header:** يحتوي على شعار الشركة، الروابط الأساسية، وزر التبديل بين اللغات.
* **Hero Section:** واجهة ترحيبية مع نص تعريفي وصورة تعبيرية.
* **Services Section:** عرض الخدمات في بطاقات (Cards) منظمة.
* **About:** قسم يشرح من نحن وأهداف الفريق.
* **Latest Projects:** معرض لأحدث الأعمال المنفذة.
* **Contact / Footer:** معلومات التواصل والروابط الهامة.

## 6. كيفية إضافة نصوص جديدة
لتحديث أو إضافة نصوص، يرجى تعديل الملفات التالية داخل مجلد `locales/`:
1. أضف المفتاح والترجمة في `ar.json`.
2. أضف نفس المفتاح والترجمة في `en.json`.
3. استخدم السمة `data-i18n="key_name"` في عنصر HTML المطلوب.
